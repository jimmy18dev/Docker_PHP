# Docker with PHP 5.6
NGINX + PHP5.6 + phpMyAdmin + MariaDB | PHP development environment generator built with Docker containers.

# Prepare project:

Clone it!
```
git clone  git@github.com:jimmy18dev/docker_php_5.6.git
```
Go to dir...
```
cd docker_php_5.6
```

Docker Compose and Dev ^^
```
docker-compose up -d
```
